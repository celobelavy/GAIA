malloc : 그냥 메모리 할당 받음, 특별한 영역 등에서 가져오지 않음
calloc : 메모리를 0으로 초기화 되어 있는 부분을 할당 받거나, 할당을 일반 malloc 받은 뒤 0으로 초기화, 크기를 받음
realloc : 이미 할당된 데이터를 새로운 영역에 할당, malloc, calloc 할당 받은 메모리의 크기 변환, 재할당 할 수는 있지만 굳이.?, 할당에 실패하면 NULL RETURN 그래서 기존 변수에 바로 relloc을 하면 안된다. 실패하면 release 되지 않는다, free를 실행하지 않는다 >> release가 되지 않았을 때, 이것을 에러로 처리해야한다면 처리하면서 수동으로 free 해야한다.
